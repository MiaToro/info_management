Description
-----------
This module can be used to manage contact cards etc..

Requirements
------------
Drupal 7.x

Installation
------------
1. Copy the entire info_management directory to the Drupal sites/all/modules directory.

2. Login as an administrator. Enable the module in the "Administer" -> "Modules"

3. (Optional) Edit the settings under "Administer" -> "Configuration"
